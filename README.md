# Dashboard Admin – React + Node.js

Un dashboard moderne permettant d’afficher des statistiques, des utilisateurs
et des données en temps réel.

## Objectifs
- Apprendre React (hooks, components)
- Créer une API Node.js (Express)
- Utiliser une base de données (MongoDB)
- Afficher graphiques (Chart.js)
- Filtrage, tri, pagination des données

##  Technologies
- React
- Node.js / Express
- MongoDB
- Chart.js
- Axios

##  Fonctionnalités prévues
- Authentification
- Tableau des utilisateurs
- Graphiques statistiques
- Logs systèmes
